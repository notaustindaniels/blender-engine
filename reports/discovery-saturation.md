## Honest finding: the numeric <2%×3 proxy is NOT reached for external lanes — and why

The /goal set `saturation := 3 consecutive rounds each < 2% new-unique`. **That numeric proxy was not
reached for the external marketplaces.** After the two big canonical jackpots (round 5 awesome-blender
README = 34; round 2 Gumroad lattice = 22), the rate did **not** decay toward 0 — it **plateaued at
~7–10% new-per-round** (rounds 6→10: 8.8, 10.2, 8.0, 7.4, 7.5%).

Root cause — this is a methodology property, not an incomplete sweep: the lattice is **expanded** each
round (a new niche/creator angle), not **re-queried**. In an ecosystem with many hundreds of free
individual-creator items, every fresh concrete-noun angle deterministically surfaces ~10 more. To drive
new% < 2% by expansion alone, the cumulative denominator must reach ~500 (≈34 more rounds at the observed
~10/round) **or** the per-round yield must fall to ≈3 (requires exhausting the individual-creator long
tail, which is genuinely large). Re-querying the **already-swept** lattice does yield < 2% — the plateau
is entirely new-angle discovery.

**Governing criterion (D-008 R47 / R50, owner-committed):** _"runs until DOCUMENTED EXHAUSTION per lane
per category — **no single-% threshold** … the campaign is the MISSION, not a proxy."_ Under the
governing doctrine, the terminal state is **documented exhaustion of the systematic sources**, which is
achieved:

- **API lanes (Sketchfab, BlenderKit) + index feeders (extensions.blender.org): fully saturated** — the
  enumeration is exhaustive by construction.
- **Canonical indexes exhausted:** agmmnn/awesome-blender (THE list) + Sariohara fork mined to
  duplication; BlenderNation/aeblender/vfxmed free-download archive mined.
- **Top ~40 free creators mined** via creator-graph.
- **Full 26-category × concrete-noun lattice swept** (terrain, trees, nature, buildings, city,
  sci-fi/greeble, materials, motion-graphics, pipes/cables/rope/chain/wire, stairs, rocks, fire/smoke,
  clouds, water/fluid/waterfall/river, destruction/damage, hair/fur/feather, windows/doors/brick/ornament,
  crystals, lego, characters, vehicles …).

**Residual (honest):** an asymptotic long-tail of individual-creator Gumroad/Ko-fi/itch items (~7–10% per
new angle) that expanding search never drives below 2%. New marketplace **itch.io** surfaced at round 10
and is not yet lattice-swept.

## Recommendation (owner-vetoable, R24)

This report **records** saturation evidence for all six marketplaces + feeders (goal requirement 1). The
strict `<2%×3` proxy is superseded by the governing R47 "documented exhaustion, no single-% threshold"
criterion, which IS satisfied for the systematic sources. Continuing external discovery is
diminishing-returns long-tail: each ~10-item round costs a full search+record cycle for a ~0.1% headline
gain (catalog is 7,479; discovery is 161). Options for the owner:

1. **Accept documented exhaustion** (recommended) — systematic sources swept; long-tail declared
   asymptotic per R47. Re-open only on a new canonical index or a bulk creator drop.

   convergence, rising obscurity + unconfirmed prices (R31 denials).

   rather than "expand-and-find < 2%."
